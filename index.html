Web Sockets

What kind of problems web sockets solve?

They enable to send data to the client and the communication with the server and between servers

In what kind of situations you should use them?

In the application that include the situation that require the multiple users communicating with each other and if the user needs to get server-side data that keeps changing.

Examples: multiplayer games, group editing, news updates.
